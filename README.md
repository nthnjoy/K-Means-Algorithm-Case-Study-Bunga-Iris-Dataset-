1. CLONE REPOSITORY
   git clone https://github.com/nthnjoy/K-Means-Algorithm-Case-Study-Bunga-Iris-Dataset-.git
   cd K-Means-Algorithm-Case-Study-Bunga-Iris-Dataset-


2. MENJALANKAN DI JUPYTER NOTEBOOK (LOKAL)
   a. pastikan python sudah terinstall
   python --version
   b. buat virtual environment (opsional tetapi direkomendasikan)
   python -m env venv
   aktifkan:
   - windows
   venv\Scripts\activate
   - mac/linux
   source venv/bin/activate
   c. install jupyter notebook
   pip install notebook
   pip install jupyterlab
   d. jalankan jupyter notebook
   jupyter notebook
